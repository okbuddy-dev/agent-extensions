# skills/wren

A Claude Code skill for agents with direct shell access to the `wren` todo CLI.

## What It Does

`SKILL.md` teaches agents how to manage wren tasks: listing, creating,
completing, reading, and scheduling tasks via the `wren` command line.

## Audience

Agents running on a machine where `wren` is installed and accessible on PATH.
For agents in locked-down environments, use `mcp-servers/wren` instead.

## Installation

Copy or symlink `SKILL.md` into your Claude Code skills directory and enable
it via your Claude Code settings.
